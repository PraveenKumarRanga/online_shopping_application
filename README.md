# online_shopping_application
The project is based on an online shopping application

The e-commerce application contains the below functionalities.
Admin Role
- Admin can log in with their credentials.
- Product management by category.

User Role
- Users can log in with their credentials.
- Users can add multiple products to the cart.
- Users can place orders.
- Users can add multiple addresses.

Product
- Multiple products can be added to related categories.
- Product can be added by admin only.

Cart 
- Cart is accessible by admin and users.
- Cart is associated with customers by one-to-one relationship.
- Cart is automatically assigned to the customer.
- Cart is capable of containing multiple products
- User can delete products from the cart.

Spring Security is used for managing the security of the application with JWT.
